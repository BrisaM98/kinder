# Sistema de gestión de Kinder
Sistema desarrollado en PHP y MySQL para la gestión de maestros, grupos y alumnos.

Registrar Usuarios: Crear la cuenta de acceso (docente).
Registrar Personal: Asignar datos personales al usuario creado.
Registrar Grupos: Crear grupos asignando un docente responsable.
Registrar Alumnos: Inscribir alumnos en los grupos existentes.

### Base de Datos
El diseño de la base de datos se encuentra en el archivo `kinder.sql`.
